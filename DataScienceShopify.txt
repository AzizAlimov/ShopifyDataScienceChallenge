`SELECT LastName
FROM (
    SELECT LastName, MAX(orderCount) 
        FROM (
            SELECT e.LastName, Count(*) as orderCount 
            FROM Orders as o 
            LEFT JOIN Employees as e 
            ON o.EmployeeID=e.EmployeeID 
            GROUP BY o.EmployeeID
    )
);` 
Peacock

We are return the last name of the employee with the most number of orders

c)
`SELECT ProductName
FROM (
    SELECT ProductName, MAX(NumOrders)
    FROM (
        SELECT p.ProductName, SUM(od.Quantity) as NumOrders 
        FROM Orders as o
        LEFT JOIN OrderDetails as od
        ON o.OrderID=od.OrderID
        LEFT JOIN Products as p
        ON od.ProductID=p.ProductID
        LEFT JOIN Customers as c
        ON c.CustomerID=o.CustomerID
        WHERE Country="Germany"
        GROUP BY ProductName
    )
);`
Boston Crab Meat

We find the orders where the customer is located in germany, group by the product then find the number of quantity shipped for each product then return the max of those.
